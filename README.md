# PDINet
Prototype-Based Diversity and Integrity Learning for All-Day Multi-Modal Person Re-Identification. TIFS 2025.

## Contribution
- A baseline method called Prototype-based Diversity and Integrity learning network (PDINet) tailored for the all-day multi-modal person ReID task. 
- A comprehensive dataset (AllDay843) to alleviate the inadequate dataset diversity, which comprises 91,371 images of 843 identities captured by multi-modal cameras across various periods throughout the day, while incorporating numerous real-world challenges.
